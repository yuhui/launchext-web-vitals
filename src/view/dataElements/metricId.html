How this Data Element works

This Data Element returns the unique ID representing this reported metric that's specific to the current web page shown in the user's web browser. Refer to the Web Vitals project to learn more about the id metric property.

This data element must be used with an Event from this Web Vitals extension.

How to use:

  • %data element name%
  • _satellite.getVar('data element name', event)
