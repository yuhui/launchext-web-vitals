How this Data Element works

This Data Element returns the name of the measured metric in acronym form. It can be one of the following:

  • CLS
  • FCP
  • FID
  • LCP
  • TTFB

This data element must be used with an Event from this Web Vitals extension.

How to use:

  • %data element name%
  • _satellite.getVar('data element name', event)
